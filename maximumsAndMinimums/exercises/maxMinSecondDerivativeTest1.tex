\begin{exercise}

The function $f(x) = x^3-27x+6$ has two critical points.  If we call
these critical points $a$ and $b$, and order them such that $a < b$,
then

$$
a = \answer{-3}
$$

$$
b=\answer{3}
$$

$f''(a)$ is \wordChoice{\choice{positive} \choice[correct]{negative}}, so $x=a$ is a local \wordChoice{\choice[correct]{max} \choice{min}}.

$f''(b)$ is \wordChoice{\choice[correct]{positive} \choice{negative}}, so $x=b$ is a local \wordChoice{\choice{max} \choice[correct]{min}}.




\end{exercise}
