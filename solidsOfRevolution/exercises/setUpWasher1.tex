	Let $R$ be the region in the $xy$-plane bounded by $y=e^{-x/2}$ and $y=e^{x/2}$ for $0 \leq x \leq \ln(3)$. Use the washer method to set up the integral(s) that give(s) the volume of the solid generated when $R$ is revolved about the line $y=3$.
