\begin{exercise}

Consider the function $\displaystyle{f(x) = \sqrt[3]{x+2}}$.  The inverse to $f$ is 
\[
f^{-1}(x) = \answer{x^3 - 2}.
\]

\begin{exercise}

Consider the function $\displaystyle{\frac{1}{x-2}}$. The inverse to $f$ is
\[
f^{-1}(x) = \answer{\frac{1}{x} + 2}.
\]

\end{exercise}
\end{exercise}
