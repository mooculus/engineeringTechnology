\begin{exercise}
This exercise verifies Clairaut's Theorem in the context of a specific example.

Suppose that $F(x,y) = x^3+4xy-6xy^3$.

Then,

\begin{itemize}
\item $F_x = \answer{3x^2+4y-6y^3}$
\item $F_y = \answer{4x-18xy^2}$
\end{itemize}

To verify Clairaut's Theorem, we have to establish the equality of the mixed partial derivatives.

Noting that $F_{xy}=$ \wordChoice{\choice{$\Pp{^2F}{x \partial y}$} \choice[correct]{$\Pp{^2F}{y \partial x} $}}, we find that 

\[
F_{xy} = \pp{y} \left[\answer{3x^2+4y-6y^3} \right] = \answer{4-18y^2}.
\]

Noting that $F_{yx}=$ \wordChoice{\choice[correct]{$\Pp{^2F}{x \partial y}$} \choice{$\Pp{^2F}{y \partial x} $}}, we find that 

\[
F_{yx} = \pp{x} \left[\answer{4x-18xy^2} \right] = \answer{4-18y^2}.
\]

Do the results agree?
\begin{multipleChoice}
\choice[correct]{Yes}
\choice{No}
\end{multipleChoice}

\end{exercise}
